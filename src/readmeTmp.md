VSCode Extension CSS Variables Snippets

## Use

```txt
# base
vscode-activityBar-border => --vscode-activityBar-border
--vscode-activityBar-border => --vscode-activityBar-border

# var
var-vscode-activityBar-border => var(--vscode-activityBar-border)
```

## Language

- css
- scss
- less
